# Arrow Fleet Manager User Guide

## Dashboard

The dashboard provides access to:

- Daily Inspections
- Vehicles
- Reports
- Settings

---

## Creating an Inspection

1. Open Dashboard
2. Select New Inspection
3. Enter vehicle details
4. Complete checklist
5. Add comments
6. Save inspection

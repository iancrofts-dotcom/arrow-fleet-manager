# Arrow Fleet Manager Roadmap

## Vision

Arrow Fleet Manager is a cross-platform fleet management system developed for
Arrow Specialised Transport.

The objective is to replace paper-based fleet inspections with a modern,
professional application capable of managing vehicles, drivers,
daily inspections, defects and compliance.

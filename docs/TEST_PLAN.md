# Arrow Fleet Manager Test Plan

## Before Every Release

### Build

- flutter clean
- flutter pub get
- flutter analyze
- flutter run

---

## Dashboard

- Opens successfully
- Statistics displayed
- Quick action buttons work

---

## Inspection Module

- Opens correctly
- Vehicle details can be entered
- Checklist works
- Comments save
- Progress displays
- Save button responds

---

## Git

- Commit created
- Push successful
- Repository clean

---

## Acceptance Criteria

Every release must:

- Build successfully
- Pass analyzer
- No runtime exceptions
- All navigation works
